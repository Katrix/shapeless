**shapeless** is a type class and dependent type based generic programming
library for Scala.

**This branch used to be the home for shapeless 2 for Scala 3. It has now moved to it's own [repository](https://github.com/typelevel/shapeless-3) in the Typelevel GitHub organisation.**

Please open issues and PRs for shapeless 3 in the new repository.
